### Usage: `vbx COMMAND [VMNAME]`
COMMANDS
* `run [VMNAME]` : run given vm or selected
* `list|ls` : List units
* `status|st [VMNAME]` : get state of all or exact vm
* `suspend|sus [VMNAME]` : suspend given vm or selected
* `stop VMNAME` : stop or discard_state of given vm
